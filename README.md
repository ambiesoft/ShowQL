# ShowQL
Shows Quick Lanuch menu

## Assign onto the Taskbar
Run with commandline '-v' or press CTRL when start running.
After MessageBox appears, pin it onto the Taskbar.

## Environment
Visual Studio 2019 runtime library are required.
